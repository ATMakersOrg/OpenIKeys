iTunes Overlay

 

The iTunes overlay is designed to operate common features found on iTunes, Apple’s popular digital jukebox. This overlay was designed to work with iTunes versions 4.9 and 5.0 for Windows and Macintosh.

 

How to Use

 

Launch iTunes and insert the iTunes overlay on your IntelliKeys USB. Press Play/Pause to begin playing the first song in your library. Press Next Song to play the next song in your library. Press Play/Pause again to pause the song. Press Previous Song to play the previous song. This overlay was designed to fit the 12 hole custom keyguard.

 

 

Special Features/Additional Keys

 

  • Navigate – navigates to open panes within iTunes.
  • Up and Down Arrows – used with the Navigate key to navigate items in the selected pane.
  • Browse – shows or hides the Genre, Artist and Album panes.
  • Show/Hide Artwork – shows or hides album artwork if present.
  • Visualizer On/Off – shows or hides the Visualizer.
  • Volume Down, Volume Up – lowers and raises the volume.

Safari Overlay

 

The Safari overlay is designed to provide access to common features found in Apple’s Web browser. It includes Previous Page, Next Page, Reload, Stop, Home, Add Bookmark, Bookmarks, AutoFill, and Search. It also includes common parts of Internet Web addresses to save keystrokes when browsing. This overlay is for Macintosh computers only. This overlay was designed to work with Safari version 2.01.

 

How to Use

 

Launch Safari and insert the Safari overlay on your IntelliKeys USB. Press the Open Location key to place your cursor at the address bar. Press the IntelliTools key. Safari will be connected to the IntelliTools Web site.

 

Special Features/Additional Keys

 

  • Open Location – places your cursor at the address bar (the text box where you enter a World Wide Web address). Type in the address and press Enter.
  • Apple.com – connects to the Apple Web site.
  • Print and Copy – make it easy to access these common commands.
  • Scroll Up and Scroll Down – moves through a page one screen at a time.
  • Tab – navigates Safari’s Address Bar and Bookmarks Bar as well as links on Web pages.
  •  Up and Down arrows – scroll a page continuously.
  • www., .com, .edu, .gov, .net, .org – inserts these common parts of an Internet Web address in the address bar.
  • AutoFill – completes web forms using information from your Address Book card.
  • Bookmarks – shows all bookmarks. Use the arrow keys to select a bookmark and press the Space key to go to the Web page represented by the selected bookmark.
  • Add Bookmarks – adds the current Web page to your bookmarks.

Kurzweil 3000 Overlay

 

The Kurzweil 3000 overlay is designed to provide access to many features of the Kurzweil 3000 word processing program. It includes large reading keys on the bottom left for easy access to this commonly used feature. This overlay was designed to work with Kurzweil 3000 version 9. This overlay will only work with the Windows version of Kurzweil 3000.

 

How to Use

 

Launch Kurzweil and insert the Kurzweil 3000 overlay on your IntelliKeys USB. Open a document and press the Read/Pause key to begin reading the document. Press the Read/Pause again to stop reading.

 

Special Features/Additional Keys

 

  • Bookmark – opens the Bookmarks dialog box. Use the Tab and arrow keys to navigate.
  • Add – adds a bookmark when the Bookmarks dialog box is open.
  • Go To – moves the cursor to the selected bookmark in the Bookmarks dialog box.
  • Word Predict – toggles Word Prediction on and off.
  • Highlight (Yellow, Green, Cyan, and Magenta) – toggles highlighting on and off.
  • Erase – toggles Erase on and off.
  • Drag Right – after pressing a Highlight or Erase key, use this key to highlight (or erase the highlight on) words to the right of the cursor.
  • Footnote – opens a text box on the bottom of the screen and creates a Footnote anchor at the cursor location. Press Close Dialog Box when you are finished typing the footnote.
  • Magnify – magnifies the spoken word.
  • Close Dialog Box – closes the Bookmark, Footnote, Spell, and Spell Check dialog boxes.
  • Close Document – closes open documents. (The Definition and Synonym keys open new documents which can be closed with this key.)

Kurzweil 3000 (Reading) Overlay

 

The Kurzweil 3000 (Reading) overlay is designed to provide simplified reading and navigation access to the Kurzweil 3000 word processing program. This overlay was designed to work with Kurzweil version 9 for Windows and Macintosh.

 

How to Use

 

Launch Kurzweil and insert the Kurzweil 3000 (Reading) overlay on your IntelliKeys USB.

On Windows computers, the Kurzweil 3000 (Reading) overlay is attached to the Kurzweil program but it is not the default overlay that sends to IntelliKeys USB. To send the Kurzweil 3000 (Reading) overlay, open the IntelliKeys USB control panel and select the Applications tab. On the left side, select Kurzweil 3000. On the right side select the Kurzweil Reading overlay. Click Done to close the IntelliKeys USB control panel. This overlay was designed to fit the 3 hole custom keyguard.

 

Special Features/Additional Keys

 

  • Read/Pause – reads the text or pauses the reading.
  • Previous Page – goes back one page.
  • Next Page – advances one page.

Microsoft Word Overlay

 

The Microsoft Word overlay is designed to provide access to many of the common features in Microsoft Word. This overlay was designed to work with Microsoft Word 2002 (version 10) and 2003 (version 11) for Windows and Microsoft Word version X and 2004 for Macintosh.

 

How to Use

 

Launch Microsoft Word and insert the Microsoft Word overlay on your IntelliKeys USB.

Press New to open a new blank document. Type a sentence.  Press Center to center the text. Press Select All to select the text. Press Bold to change the text style to bold. Press the Spelling key to check the spelling and grammar in the document. Use the Tab and Arrow keys to navigate the Spelling and Grammar window, and use the Enter key to make your selection. Press the Close Dialog Boxes key to close the spelling Window.

 

Special Features/Additional Keys

 

·         Header/Footer – places the cursor in the document’s header. Use the Down Arrow to go to the Footer. Press the Header/Footer key again to return to the document.

·         AutoText – shows a list of AutoText entries including common salutations and closings. Use the Up and Down arrows to navigate the list and the Enter key to add the selection to your document.

·         Font – opens the font dialog box. Press tab to go from Font to Font Style, Size, Color, etc. Use the Up and Down arrows to scroll through the selections. Use Enter to accept your selections and close the Font dialog box.

·         Bullets/Numbers – opens the Bullets and Numbering window. Use your Arrow keys to choose a bullet style, and then press the Enter key to add the bullets to your document.

·         Close Dialog Boxes – closes most dialog boxes including Print, Find, and Spelling.

TestTalker Overlay

 

The TestTalker overlay provides access to Multiple Choice and True/False tests created in Freedom Scientific’s TestTalker. This overlay is for use on Windows computers only and was designed to work with TestTalker version 1.1.

 

How to Use

 

Launch TestTalker student edition, open a Multiple Choice or True/False test, and insert the TestTalker overlay on your IntelliKeys. Use the Read key to read the first question. Press the Read key again to pause. Press the Next Answer key to navigate to the answer you wish to choose. Press the Select key to mark your answer.

Note: For Fill-in-the-Blank tests that require keyboard input, you can use the WYNN overlay which is already attached to the TestTalker application. On the Applications tab in the IntelliKeys USB control panel, select TestTalker from the list of applications. From the list of overlays on the right side, select the WYNN overlay and click done.  All of the keys on the TestTalker overlay are also available on the WYNN overlay. Use Tab to navigate to the next answer and Shift+Tab to navigate to the previous answer. Use Space to select an answer.

This overlay was designed to fit the 9 hole custom keyguard.

 

Special Features/Additional Keys

 

  • Save – saves the document.
  • Print – prints the document.
  • Previous Page – goes back one page.
  • Next page – goes forward one page.
  • Read – Toggles Reading on and off.
  • Last Answer – highlights the previous answer.
  • Next Answer – highlights the next answer.
  • Select – selects the highlighted answer.

WYNN Overlay

 

The WYNN overlay is designed to provide access to all of the toolbar items in Freedom Scientific’s WYNN literacy software. This overlay is for use on Windows computers only. This overlay was designed to work with WYNN 3.51 and WYNN 4.

 

How to Use

 

Launch WYNN and insert the WYNN overlay on your IntelliKeys USB. Press the Next key to bring the focus to the Next button in WYNN. Press the Switch Toolbar key to change to the next color-coded toolbar. Use the Arrow keys to move to the button you want. Press the Select key to activate the button. Use the Tab key to navigate the buttons in open tools and the Space key to activate the buttons.

 

Special Features/Additional Keys

 

  • Read/Pause – toggles reading on and off.
  • Write – toggles between Edit Mode and Read-Only Mode.
  • Exact View – toggles between Exact View and Text View.
  • Scan – begins scanning a page placed on your scanner.
  • Go To Web – opens WYNN’s Web browser.
  • Close Web – closes WYNN’s Web browser.
  • Next – highlights the Next Button.
  • Switch Toolbar – cycles through the toolbars when the Next button is active.
  • Arrow Keys – navigate to the various toolbar buttons.
  • Select – activates the selected toolbar button.
  • Previous Page – goes back one page.
  • Next Page – goes forward one page.
  • Word Prediction – toggles Word Prediction on and off.
  • Function keys (F1-F6) – use in conjunction with Word Prediction to select words.

AOL Explorer Overlay

 

The AOL Explorer overlay is designed to provide access to common features found in AOL’s Web browser. It includes Back, Forward, Stop, Refresh, Favorites and Add to Favorites. It also includes common parts of Internet Web addresses to save keystrokes when browsing. This overlay was designed to work with AOL Explorer version 1.1 for Windows only.

 

How to Use

 

Launch AOL Explorer and insert the AOL Explorer overlay on your IntelliKeys USB. Because AOL Explorer is installed to a unique location on every computer, you will need to find the AOL Explorer application. Open the IntelliKeys USB and IntelliSwitch Control Panel and click the Applications tab. In the list of applications, click AOL Explorer (not found), and then click Find. Browse to AOLServiceHost.exe and click Open. The path to the application will be similar to: c:\Program Files\Common files\AOL\1127266671\ AOLServiceHost.exe

The numbered folder in this path will be unique on each computer.

 

Special Features/Additional Keys

 

  • Open Location – places your cursor at the address bar (the text box where you enter a World Wide Web address). Type in the address and press Enter.
  • AOL.com – connects to the AOL Web site.
  • Print, Copy, and Paste – make it easy to access these common commands.
  • Scroll Up, Scroll Down – moves through a page one screen at a time.
  • Tab – navigates the Address Bar and links on the web page.
  •  Up and Down arrows – scroll a page continuously.
  • www., .com, .edu, .gov, .net, .org – inserts these common parts of an Internet Web address in the address bar.
  • Favorites – shows a list of all favorites. Use the arrow keys to select a favorite and press the Enter key to go to the Web page represented by the selected favorite.
  • Add to Favorites – adds the current Web page to your Favorites.

Co:Writer Overlay

 

The Co:Writer overlay is designed to provide access to Don Johnston’s word prediction and vocabulary assistant program. This overlay was designed to work with Co:Writer Solo Edition, version 5.0 for Windows and Macintosh.

 

How to Use

 

Launch Co:Writer and connect it to your Word processing program. Insert the Co:Writer overlay on your IntelliKeys USB. Begin typing and use the number keys to choose a word listed. Now press the Send key. The word will be typed in your word processor.

Note for Windows users: If you connect to Write:OutLoud, which already has an overlay attached, you may want to set the IntelliKeys USB control panel to send no overlay for Write:OutLoud. Open the IntelliKeys USB control panel. Click the Applications tab. Select Write:OutLoud in the application list, then click Send No Overlay. Click Done.

Note for Mac users: The Co:Writer overlay will not send automatically. You will need to attach the Co:Writer overlay to the word processor you use. See the section on Attaching Overlays to Applications. If you connect to Write:OutLoud, the Co:Writer overlay is already attached. All you need to do is to select the Co:Writer overlay as the overlay to send for Write:OutLoud on the Applications tab of the IntelliKeys USB control panel.

 

Special Features/Additional Keys

 

  • Show and Hide – shows or hides the Co:Writer window.
  • True – lets you type numbers instead of choosing a word in the prediction list.
  • More – shows more word choices in the word prediction list.
  • Prefix and Suffix – shows more word choices with prefixes or suffixes added.
  • Speak Sentence – speaks the current sentence.
  • Send – sends the text in the Co:Writer window to your word processor.
  • Undo – cancels the last change you made.
  • Erase – erases the previous word.

Write:OutLoud Overlay

 

The Write:OutLoud overlay is designed to provide access to commonly used features of Don Johnston’s talking word processor. The overlay includes keys for navigation, speech, spell check, and dictionary. This overlay was designed to work with Write:OutLoud Solo Edition version 4.0 for Windows and Macintosh.

 

How to Use

 

Launch Write:OutLoud, open a document, and insert the Write:OutLoud overlay on your IntelliKeys USB. Press Select Sentence to select the sentence to the right of the cursor. Press Speak to have the selected sentence read.

Note: If you use Co:Writer with Write:OutLoud, see the overlay description for Co:Writer.

 

Special Features/Additional Keys

 

  • Save and Print – make it easy to access these common commands.
  • Beginning – moves the cursor to the beginning of the document.
  • Next Sentence – moves the cursor to the beginning of the page.
  • Select Sentence – Selects the sentence to the right of the cursor.
  • End – moves the cursor to the end of the document.
  • Speak – Speaks the current sentence.
  • Repeat – repeats the last sentence spoken.
  • Spell Check, Dictionary, Homonym, Add Picture, and Bibliographer keys all open new windows.  Use the Tab and arrow keys to navigate, and the Space key to select in these windows. 
  • Write More – adds two blank lines after the cursor to give you room to write more.
  • Add Picture – opens the Picture Library window.

Inspiration Overlay

 

The Inspiration overlay is designed to provide access to most features of Inspiration, the premier tool to develop ideas and organize thinking. This overlay was designed to work with Inspiration version 7.6 for Windows and Macintosh.

 

How to Use

 

Launch Inspiration and insert the Inspiration overlay on your IntelliKeys USB. Press the Diagram/Outline key to toggle between Diagram View and Outline View. In Diagram View you can use any of the keys in the green area to create and arrange symbols. In Outline View you can use any of the keys in the yellow area to create and arrange topics. The keys in the orange area will work in both views.

 

Special Features/Additional Keys

 

Diagram View Keys

  • RapidFire – turns on RapidFire mode to quickly create new symbols.
  • Arrange – opens the Arrange diagram window. Use tab and arrow keys to navigate.
  • Fit to Window – sizes the diagram so it will fit on the screen.
  • Show/Hide Subtopic – shows or hides the selected subtopic.
  • Scroll to Selection – centers the selected symbol.
  • 5 shape keys – change the shape of the selected symbol.
  • Create Left, Right, Up, Down – creates a new symbol in the direction shown. You can begin typing in the symbol as soon as it is created. Press Cancel/Exit Text Editing when you are done typing.

 

Outline View Keys

  • Topic – adds a new topic.
  • SubTopic – adds a new subtopic to the selected topic.
  • Move Left – moves the selected topic left.
  • Move Right – moves the selected topic right.
  • Flip Up – moves the selected topic up in the outline order.
  • Flip Down – moves the selected topic down in the outline order.
  • Promote – moves all subtopics to the level of the selected topic.
  • Demote – makes all topics below the selected topic into subtopics.
  • Show/Hide Subtopic – shows or hides the selected subtopic.

 

Other Keys

  • Diagram/Outline – toggles between Diagram View and Outline View.
  • Cancel/Exit Text Editing – exits text editing and RapidFire modes.
  • Note – toggles note on and off.
  • Spell – checks the spelling in your document.
  • Zoom Out, Zoom In – makes the diagram or outline larger or smaller.
  • Save – saves the current document.

Kidspiration Overlay

 

The Kidspiration overlay is designed to provide access to the most common features of Kidspiration, the visual learning tool for K–5 learners. This overlay was designed to work with Kidspiration version 2 for Windows and Macintosh.

 

How to Use

 

Launch Kidspiration and insert the Kidspiration overlay on your IntelliKeys USB. Press the Pictures/Writing key to toggle between Picture View and Writing View. In Picture View you can use any of the keys in the green area to create and arrange your picture. In Writing View you can use any of the keys in the yellow area to create and arrange your ideas. The keys in the orange area will work in both Views.

 

Special Features/Additional Keys

 

Picture View Keys

  • Starter – exits current document and returns to Starter menu. Use Tab to navigate the Starter menu and Enter to select.
  • Link Symbols – creates a new link. Use the arrow keys to move the link to a symbol and then press Enter.
  • SuperGrouper – adds a SuperGrouper shape to the diagram. Press the SuperGrouper key again to toggle through the default shapes.
  • Shape keys – change the shape of the selected symbol.
  • Create Left, Right, Up, Down – creates a new symbol in the direction shown.

 

Writing View Keys

  • Add idea – adds an idea in Writing View.
  • Move Left – moves the selected idea left.
  • Move Right – moves the selected idea right.

 

Other Keys

  • Pictures/Writing – toggles between Picture View and Writing View.
  • Cancel/Exit Text Editing – Use this key to exit text editing mode.
  • Spell Check – checks the spelling in your document.
  • Clear – removes the selected symbol or idea.
  • Undo – cancels the last change you made.
  • Student Name – opens the Student Name dialog box.
  • Save – saves the current document.
